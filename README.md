# 📚 Academia Library Management System

<img width="4000" height="3000" alt="LMS" src="https://github.com/user-attachments/assets/3d25192f-2852-483f-b2dc-03d156c01caf" />


A modern and responsive **Library Management System** built using **React + TypeScript + Vite**.  
This project helps students, employees, and administrators manage books efficiently with an attractive UI and smart features.

---

## 🚀 Features

### 👨‍🎓 Student Portal
- Browse available books
- Search books instantly
- View recently added books
- Book details popup/modal
- Responsive dashboard UI

### 👨‍💼 Employee Portal
- Add new books
- Maintain latest 5 added books
- Manage library records
- Update book availability

### 📊 Analytics Dashboard
- Monthly popular books histogram/chart
- Track book popularity
- Visual data representation

### 🌐 General Features
- Modern responsive design
- Smooth navigation
- Login page
- Interactive components
- Reusable React components
- Fast performance using Vite
